# Emerging Technologies Group Research Project

Using HTML and CSS

## Project Outline

You will research up to 4 emerging computer technologies at a superficial level, explaining what the technologies are and their impacts on a variety of societies and cultures. Research one more emerging technology in more detail to identify 2 positive and 2 negative impacts on society or the economy. Create web pages to report your research and combine your pages with 1 or 2 other teammates (total size of group is 2-3 people) to make a larger emerging technologies website. Coordinate with others so that your emerging technologies do not overlap. This project is designed so that there should be no merge conflicts. Include your research sources.

## Emerging Technology

An emerging computer technology is a technology that is still being developed and greatly improved. (There is overlap with theoretical computing and collaborative industries – you may use your previous research/answers for this assignment). Examples of emerging computer technologies include:

- Artificial intelligence
- Generative AI
- Agentic AI
- Physical AI
- Computer Vision
- Quantum computing
- Neuromorphic computing
- Edge computing
- Confidential computing
- Industrial Internet of Things (IoT)
- Extended Reality (XR)
- Brain-computer interfaces (BCI)
- Multimodal User Interfaces
- Post-quantum Cryptography
- Blockchain
- Green Technology
- DNA-based data storage
- Bioinformatics

## Research Requirements

Research 4 emerging technologies at a superficial level. There cannot be any research topic overlap between anyone in the same group.
For each of the 4 technologies, give a short 50-75 word description of the technology. As part of this description, state one impact of the technology. For the best mark, consider impacts on different people:

- One technology’s impact should be about the effects on most of Canadian society.
- One technology’s impact should be about the effects on a minority in Canadian society.
- One technology’s impact should be about the effects on world economy.
- One technology’s impact should be about the effects on a developing society.

Research one additional technology more intensively. Explain what your technology is. Discuss 4 impacts of your technology: 2 positive impacts and 2 negative impacts; one of the impacts should be the economic impacts. In this discussion, consider the impacts of the technology on different societies and cultures, in Canada and around the world. Include images and videos. Link your research sources within your text. Format this report using CSS to adjust the layout, fonts and colours.

## Reporting Requirements

Use HTML and CSS to make a nicely formatted report that has multimedia elements like images and video. A report is easier to read and follow when there are appropriate titles and headings. Please read the rubric below for the most detailed set of marking requirements.

## Teamwork Requirements

Create a shared document to show which technologies each person is researching at a superficial level.

- Create a shared repository on GitHub so each person can contribute to the full site.
- Each person should create their own branch of the repository and merge it into the main branch. Since each person is making their own unique research files, there should not be any merge conflicts here.
- There should be an index.html file in the root directory of the repository. Each person should edit this page to link to their own research pages. This file can be very simple. This index file should be made before anyone clones the repository to prevent merge conflicts. The index.html file should look like this:

> # Emerging Technologies Group Project
> 
> - Person 1 Name [links to person1Name.html](about:blank)
> - Person 2 Name [links to person2Name.html](about:blank)
> - Person 3 Name [links to person3Name.html](about:blank)
> - Person 4 Name [links to person4Name.html](about:blank)
